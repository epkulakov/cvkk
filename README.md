
🔬 **Проект посвящён разработке автоматизированной системы визуального контроля качества (СВКК) печатных плат на основе нейросетевых технологий.**

❗ **Актуальность** обусловлена низкой эффективностью ручной проверки: 
* 📉 точность $60–90\%$;
* 💰 высокая стоимость труда;
* 🧑‍🦱 субъективность и утомляемость оператора.

🎯 **Цель работы** — разработать автоматизированную систему визуального контроля качества (СВКК) печатных плат на базе нейросетевых технологий **с точностью (Precision) $95\%$ до 27.03.2026**.

🔎 Был проведён сравнительный анализ человеческого контроля, СВКК и промышленного аналога **ML Sense**. Для обнаружения дефектов была выбрана модель **YOLOv8m** из‑за баланса скорости и точности.

🛠️ Модель дообучена на датасете **PCB Defects** ($693$ изображения) методом *fine‑tuning* на $100$ эпохах. Достигнута:
* ✅ точность (Precision) — **$98\%$**;
* ✅ полнота (Recall) — **$95{,}4\%$**, что превышает поставленную цель.

💻 Разработан **веб‑интерфейс** на библиотеке **Gradio** с четырьмя вкладками:
1. 🖼️ ручная загрузка изображений;
2. 🎥 детекция через веб‑камеру;
3. 📄 формирование отчёта о дефектах;
4. 🔎 подробности проекта.

🚀 Выполнено развёртывание на сервере **VK Cloud**.

✨ **Уникальность**:
* 🚫 система **не требует сравнения с эталоном** — напрямую выявляет дефекты на любом изображении;
* 🔄 адаптируется под конкретное производство сбором целевого датасета;
* 💸 экономически целесообразна (**аренда сервера ~5000 руб/мес** против **зарплаты контролёра ~68 000 руб/мес**).
